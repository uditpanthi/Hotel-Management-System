# 🏨 CozyHaven - Hotel Management System

CozyHaven is a full-stack hotel booking and management application built using ASP.NET Core Web API and React.js. The platform enables customers to search and book hotels, hotel owners to manage their properties, and administrators to oversee the entire system through role-based access control.

## 🚀 Features

- User Registration & Login
- JWT Authentication & Role-Based Authorization
- Admin, Hotel Owner, and Customer Modules
- Hotel & Room Management
- Hotel Search & Filtering
- Room Booking & Reservation
- Review & Rating System
- Payment Management
- Responsive User Interface
- RESTful API Architecture

## 🛠 Tech Stack

- C#
- ASP.NET Core Web API
- React.js
- Entity Framework Core
- SQL Server
- Azure App Service
- Azure SQL Database
- GitHub Actions

## 🔐 Authentication

The application uses JSON Web Token (JWT) authentication with role-based authorization for:

- Admin
- Hotel Owner
- Customer

## ☁️ Deployment

- Backend: Azure App Service
- Frontend: Azure Static Web Apps
- Database: Azure SQL Database
- CI/CD: GitHub Actions
